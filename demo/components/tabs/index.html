Tabs

Segmented control for switching between related views. Uses native radio inputs so it works without JavaScript.

Default

Three tabs, first selected. Try arrow keys.

Two options

Works down to two — the minimum useful size.

Markup

Copy this into your project.

<div class="tabs" role="tablist">
  <label class="tabs__tab">
    <input type="radio" name="group" class="tabs__radio" checked />
    <span class="tabs__label">Overview</span>
  </label>
  <label class="tabs__tab">
    <input type="radio" name="group" class="tabs__radio" />
    <span class="tabs__label">Details</span>
  </label>
</div>
