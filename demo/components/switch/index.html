Switch

Binary on/off control. Uses <input type="checkbox"> under the hood so it works with forms and assistive tech out of the box.

Default

Off state — the resting default.

Checked

On state — knob shifts right, track fills with --primary.

Disabled

Non-interactive — 50% opacity, cursor blocked.

Markup

Copy this into your project.

<label class="switch">
  <span class="switch__input-wrapper">
    <input type="checkbox" class="switch__input" />
    <span class="switch__knob"></span>
  </span>
  <span class="switch__label">Notifications</span>
</label>
