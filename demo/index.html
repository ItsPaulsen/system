Demo project

A design system for a single project. One CSS file, one page per foundation, tokens ready to drop in.

What's inside

Colors Base, hover, active per family Typography Headline, Title, Body, Caption Grid Columns, gutters, margins Spacing 4px base scale Radii Corner rounding scale Shadows Elevation depth scale Motion Durations and easings

How to use

Everything ships in one tokens.css. Import it, reference with var(--…). 

@import url("tokens.css");

.card {
  background: var(--background-primary);
  border-radius: var(--radius-md);
  box-shadow: var(--shadow-sm);
  padding: var(--space-24);
}
